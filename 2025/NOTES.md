# Notes

- Installing Deno
  - `curl -fsSL https://deno.land/install.sh | sh`

- Using Deno
  - TS works out of the box
  - Format files: `deno fmt`
  - Lint files: `deno lint`
  - Run files: `deno run file.ts`

- Using Neovim
  - space s f: search for files
  - gc: comments lines
  - K: shows type
